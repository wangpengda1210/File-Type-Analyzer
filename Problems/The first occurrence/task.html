The first occurrence

The described substring searching algorithm always returns a binary result: true if a substring is found and false otherwise. In this task, you need to extend the algorithm to return the index of the first substring occurrence.

Input: two lines, the first one is a pattern, the second one is a text. It is guaranteed that the length of the text is always greater than $0$.

Output: the index of the first occurrence of the pattern in the text. If there are no occurrences, output $-1$.

Keep in mind that the empty string is a substring of any string. See the third test case as an example of how an empty pattern should be processed.


Sample Input:
aca
acbacad

Sample Output:
3


Sample Input:
banas
bananas

Sample Output:
-1


Sample Input:

abc

Sample Output:
0



Memory limit: 256 MB
Time limit: 8 seconds
