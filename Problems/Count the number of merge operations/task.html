Count the number of merge operations

Given an array of integers. Output the number of merge operations to solve the array using the standard top-down merge sort.

The first line of the input contains $N$ which is the number of elements in the array. The second line contains the array you need to sort.


Sample Input:
9
1 4 7 6 8 4 5 3 2

Sample Output:
8



Memory limit: 256 MB
Time limit: 8 seconds
